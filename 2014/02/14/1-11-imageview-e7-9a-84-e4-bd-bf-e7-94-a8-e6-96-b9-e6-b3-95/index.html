二、什么是ImageView
1.在XML文件当中使用标签

在drawable中的图片会含有id
直接将图片放置到drawable文件夹

1）建立一个ImageView

三、神奇的ScaleType
1）Center
2）CENTER_CROP
3)CENTER_INSIDE
4)FIT_CENTER(START,END)
5)FIT_XY
fit_center:把图片放大/缩小边框同样大小，居中显示
fit_end
fit_start
center:如果图片比边框大，图片只能被显示出一部分，否则，居中显示
inside：等比例的放大、缩小以放入imageview当中
CenterCrop：将图片进行等比例缩放，图片短边与imageview相等
center：将图片摆放在中央
fit：分成两组
java文件中进行设置： imageView1.setScaleType(ScaleType.CENTER);
使用imageView中要使用标签
推荐使用png

package com.dream.s01_e11_imageview;

import android.os.Bundle;
import android.app.Activity;
import android.view.Menu;
import android.widget.ImageView;
import android.widget.ImageView.ScaleType;

public class MainActivity extends Activity {
    ImageView imageView1;
    ImageView imageView2;
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);
        imageView1 = (ImageView)findViewById(R.id.imageView1Id);
        imageView2 = (ImageView)findViewById(R.id.imageView2Id);
        imageView1.setScaleType(ScaleType.CENTER);
    }

    @Override
    public boolean onCreateOptionsMenu(Menu menu) {
        // Inflate the menu; this adds items to the action bar if it is present.
        getMenuInflater().inflate(R.menu.main, menu);
        return true;
    }

}
