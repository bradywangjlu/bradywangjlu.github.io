1.9 多选按钮

Android未分类

 

一、主要内容

  1. 多选按钮（CheckBox）的基本概念

  2. <CheckBox/>与CheckBox

  3. OnClickListener与OnCheckedChangeListener监听器

二、什么是CheckBox

1.在XML文件当中使用<CheckBox/>标签

2.在代码中使用CheckBox的对象表示

3.常用OnClickListener与OnCheckedChangeListener监听器

Onclicklistener：调用参数View指的是调用该参数的对象

通过Id找控件：findVIewById

通过控件找id：getId

checkBox.isChecked（）——该按钮是否被选中

oncheckedChangeListener：有返回值确定是否被选中

public void onCheckedChanged(CompoundButton buttonView, boolean isChecked)

buttonView：传进来的View

isChecked：选中是否选中

onclickedlistener:被点击触发状态

onCheckedChangeListener：选中状态改变的时候才触发

    class checkBoxListener implements OnCheckedChangeListener{

        @Override
        public void onCheckedChanged(CompoundButton buttonView,
                boolean isChecked) {
            // TODO Auto-generated method stub
            if(buttonView.getId()==R.id.datId)
                System.out.println("eatID");
            else if(buttonView.getId()==R.id.sleepId)
                System.out.println("sleepId");
            else if(buttonView.getId()==R.id.dotaId)
                System.out.println("dotaId");
            else if(buttonView.getId()==R.id.allSelect){
                eatBox.setChecked(allSelected.isChecked());
                sleepBox.setChecked(allSelected.isChecked());
                dotaBox.setChecked(allSelected.isChecked());
                if(allSelected.isChecked())
                    allSelected.setText("取消全选");
                else
                    allSelected.setText("全选");
