info page和man page的区别在于info page 将文件数据拆成一个一个的段落，每个段落用自己的页面来撰写，并且在各个页面中还有类似网页的“超链接”跳到不同的页面中，每个独立的页面也称为一个节点 快速查询命令作用的方法：whatis/man f

查看当前在线用户：who 查看网络联机状态：netstat 查看后台运行进程：ps-aux sync：将默认缓存在内存当中的数据保存至硬盘 几种关机方法： 1.shutdown 
NAME
       shutdown - bring the system down

SYNOPSIS
       shutdown [OPTION]...  TIME [MESSAGE]
OPTIONS
       -r     Requests  that  the system be rebooted after it has been brought
              down.

       -h     Requests that the system be either halted or powered  off  after
              it has been brought down, with the choice as to which left up to
              the system.

       -H     Requests that the system be halted after  it  has  been  brought
              down.

       -P     Requests  that  the  system  be  powered  off  after it has been
              brought down.

       -c     Cancels a running shutdown.  TIME is  not  specified  with  this
              option, the first argument is MESSAGE.

       -k     Only  send  out  the warning messages and disable logins, do not
              actually bring the system down.
常用的几个参数： shutdown -h：在系统服务停掉之后关机 shutdown -r：在系统服务停掉之后重启 shutdown -t+时间：在一定时间之后执行 几个关于shutdown的例子： 
#shutdown -h now
立即关机，其中now相当于时间为0
#shutdown -h 20:25
在今天的20:25分会关机，若在21:25分之后执行该命令，则隔天才会关机
#shutdwn -h +10
在十分钟之后关机
#shutdown -r +30 'The system will reboot'
过30min系统重启，并且将后面的消息给所有在线的用户
**注：时间是必须添加的参数** 2.reboot，poweroff，halt 切换执行等级 init * run level 0：关机 * run level 3：纯命令行模式 * run level 5：带图形界面模式 * run level 6：重启 
cd：切换目录
pwd：显示当前目录
mkdir：新建一个空的目录
rmdir：删除一个空的目录
pwd 是Print Working Directory的缩写，可以使用pwd找到当前的工作目录 
dream@dream-Inspiron-5520:~$ pwd
/home/dream
mkdir创建目录的时候只能单层单层的创建，如果一次要创建多层目录，需要使用-p参数 
dream@dream-Inspiron-5520:~$ mkdir test
dream@dream-Inspiron-5520:~$ mkdir test/test1/test2
mkdir: 无法创建目录"test/test1/test2": 没有那个文件或目录
dream@dream-Inspiron-5520:~$ mkdir -p test/test1/test2
ls：查看文件和目录 -a 全部的文件，包括隐藏文件和. 以及.. -A 全部的文件，不包括.和.. -d：只列出目录本身，不列出目录内的文件 -f：只列出文件结果，不进行排序 -F：根据文件、目录等信息给予附加数据结构 *-可执行文件 /:目录 =：socket文件 |：FIFO文件 -h：将文件大小用人类容易读的方式表示 -i：写出inode号码 -l：表示文件的详细信息 -n：写出UID和GID，而不是用户名和用户组的名称 -R：连同子目录一起列出来 -S：以文件大小排序 -t：以时间排序 cp：对文件进行复制 cp 参数 源文件 目标文件 -a：相当于-pdr 复制文件的属性且进行递归复制 -i：若源文件已经存在，则进行询问 -r：进行递归调用复制 如果使用默认的话，用户的所有者和用户组就会变成当前的用户组 删除一个文件夹，同样也可以使用rm -r命令进行删除 basename：查看文件名 dirname：取得目录名 
dream@dream-Inspiron-5520:~$ basename /etc/sysconfig/network
network
dream@dream-Inspiron-5520:~$ dirname /etc/sysconfig/network
/etc/sysconfig
cat：查看文件内容 -n 打印出行号，空白行也打印行号 -b 打出行号，仅对非空白行进行显示 -A 连特殊字符也进行显示 tac：反向显示 nl：添加行号进行显示 more：一页一页翻动 more中的命令

/字符串：向下查询该字符串

touch：修改文件时间或者创建新文件 chattr:设置文件的隐藏属性 a：设置a之后，文件智能增加数据，而不能删除也不能修改数据，只有root才能设置这个属性 i：可以让一个文件不能被删除、改名、设置连接也无法写入或添加数据，只有root才能设置这个属性。 
root@dream-Inspiron-5520:~# touch test
root@dream-Inspiron-5520:~# chattr +i test 
root@dream-Inspiron-5520:~# rm test
rm: 无法删除"test": 不允许的操作
root@dream-Inspiron-5520:~# chattr -i test 
root@dream-Inspiron-5520:~# rm test
